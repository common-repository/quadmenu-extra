Create a Mega Menu in Extra.

== Description ==
[Demo](http://quadmenu.com/extra/) | [Documentation](http://quadmenu.com/documentation/) | [Developers](http://quadmenu.com/documentation/developers/) | [Community](https://www.facebook.com/groups/quadlayers/)

This WordPress Plugin allow you to create a Mega Menu in Extra and integrates QuadMenu with the Extra theme settings. Requires QuadMenu and Extra.

Customizer

https://www.youtube.com/watch?v=lYy0Fv3ZtE8

== Screenshots ==

Demo Site: http://quadmenu.com/extra/

1. QuadMenu Mega Menu working with Extra
2. This plugin adds a preconfigured "Extra" theme to fit QuadMenu Mega Menu settings.

== Installation ==

1. Go to the Plugins page in WordPress
1. Search for "Extra Mega Menu"
1. Click "Install"

After installation go to Appearance > Menus and create your mega menu.
